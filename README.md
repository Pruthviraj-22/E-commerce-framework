# E-Commerce Automation Framework

Selenium automation framework for ecommerce website testing using Python and PyTest.

## Tech Stack
- Python
- Selenium
- PyTest
- POM Framework
- HTML Reports

## Features Automated
- Login
- User Registration

## Project Structure
- pages
- tests
- utils
- reports

## Run Tests
pytest -v

## Generate HTML Report
pytest --html=reports/report.html
